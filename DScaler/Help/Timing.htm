Timing Settings
 

Settings specific to flip timing

Setting Default Description
Auto Format Detect ON Should we try and spot input format changes
50Hz Format PAL Mode to use when 50Hz input is detected
60Hz Format NTSC Mode to use when 60Hz input is detected
Format Change Threshold 2 Number of ten field periods when the timing pattern must be repeated before switching.
Do JudderTerminator On Video Modes OFF If global JudderTerminator is on then should it be used in video mode.
Sleep Interval 0 Number of ms to sleep when waiting.
Sleep Skip Fields 0 Can be used to only sleep every so many frames.
Sleep Skip Fields Late 0 Can be used to only sleep every so many frames when running late.
