What's New
 

DScaler - version 4.1.7

What's new:

  • DScaler can work on multiple-monitors
  • Black and white filter
  • Y/C shift filter
  • preview mode for TV channels allowing the display of several channels at screen (with only one active)
  • preview mode for stills
  • new option in the general hardware setup dialog box to give priority to other software
  • limit (parameter driven) size of memory allocated for stills

What's Improved

  • Settings management (including editing strings)
  • MoComp2 deinterlace mode
  • TomsMoComp deinterlace mode
  • WSS / CC decoding with SAA713x cards
  • Calibration procedure for SAA713x cards
  • Access denied error with the DScaler driver on localized versions of Win2k/XP.
  • Lots of bug fixes

What's Experimental:

  • PowerStrip modeline option when changing resolution in the full screen mode
  • DirectShow support for tuners and file
  • Skins & Toolbar
  • Timeshift
  • Resolution changing
  • Colorimetry filter
  • DScope video filter
