What's New
 

DScaler - version 4.0.0

What's new:

  • Support of multiple BT cards
  • Optimized field access code for fewer dropped frames
  • New Adaptive Noise Filter
  • New Gradual Noise Filter
  • New Histogram Filter
  • Flickerless OSD
  • Teletext support improved
  • Additional Teletext languages
  • Action added to search and lock the correct film mode
  • Sharpness Filter now works with older (MMX/3DNOW) computers
  • Some minor improvements to Temporal Comb Filter

What's Experimental:

  • New audio code
  • Support of DirectShow filters supporting AVI files
  • Support of TIFF files
  • Playlist for stills
  • Set of patterns to setup the overlay and the display
  • Direct access to the stills taken during the current session
