Greedy 2 Frame Method
 

This method tries to combine the best features of the 2-frame algorithm with the looking ahead ability of the greedy method.  I find this methods works well on sports and other high motion material.  With static images and low quality sources there may be unacceptable levels of bobbing.

This method uses four fields and has a configuarable motion detect. Motion is detected on a two pixel basis.

Who Should Use It?

Everybody. This is the current default method for faster processors as it gives the best all round performance.

Settings

Setting Default Description
Greedy 2 Frame Luma Threshold 4 Luma differences above this will be interpreted as movement.
Greedy 2 Frame Chroma Threshold 8 Chroma differences above this will be interpreted as movement.
