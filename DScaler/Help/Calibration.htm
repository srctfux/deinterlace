Calibration
 

What is Calibration ?

The calibration is a proceduce using test patterns which allows to find the best video settings for the capture card.

DScaler is able to analyze certain test patterns from commercial DVD and then adjust the video settings of the capture card so that the output signal from the capture card is the closest as possible with the real content of the test pattern stored on the disc.

How it works ?

The user has to do the following actions :

  1. play one of the referenced test patterns in his DVD player,
  2. press pause with the remote control to be sure to have the same signal during the calibration,
  3. select the corresponding test pattern in the DScaler Hardware menu,
  4. start the calibration choosing between a manual and an automatic calibration; (the commands are in the DScaler Hardware menu),
  5. if a manual calibration was chosen, adjust the different video settings,
  6. Stop the calibration when finished.

The user must adjust the video settings in this order :

  1. brightness,
  2. contrast,
  3. color (saturation U + saturation V),
  4. hue.

There are three types of test patterns used by DScaler :

  1. test patterns that content only range of grey including black and white,
  2. test patterns that content only generic colors (red, green, blue, yellow, cyan, magenta, ...),
  3. test patterns whose content is a merge of the two precedent types.

the first type of test pattern is used to adjust only brightness and contrast.
The second type of test pattern is used to adjust only color and hue.
The third type of test pattern allows to do a full adjustment of all video settings.

Without a test pattern of the third type, the user has to do the calibration in two steps :

  • first adjust contrast and brightness using a test pattern contaning range of gray,
  • second adjust color and hue using a test pattern containing generic colors.

The user can choose between a manual calibration and an automatic calibration.
During a manual calibration, the user must adjust by hand the video settings using keyboard shortcuts or the video adjustments dialog box. An OSD screen is automatically displayed to help the user to decide what to do when the user starts the calibration. For each analyzed zone of the test pattern (having a unique color) is displayed the current differences for R,G,B and Y,U,V components between the current signal and the reference signal. The user should try to reduce these differences as much as possible; the ideal should be to have 0 everywhere.
During an automatic calibration, the user has nothing to do, just waiting that DScaler finds the best settings. An OSD screen is displayed allowing to see the progress of the operation. At the end, the user can see the new video settings, and a quality indicator for each analyzed zone of the test pattern shows how good is the card calibration for this zone.

What do you need ?

You need at least one of the referenced DVD and a DVD player connected to the capture card to play the test patterns.

You can even use a LD player if you have the Video Essentials LaserDisc.

What are the referenced test patterns ?

DVD Collector Les Années Laser 2000

A range of gray test pattern - title 36 chapter 1
A color test pattern - title 31 chapter 1

THX Optimode (Pal)

Screen Performance test pattern

You can find this test pattern on several Pal THX DVD. For instance, "Les Rivières Pourpres" in region 2.

THX Optimode (Ntsc)

Monitor Performance test pattern
Tint Color test pattern

You can find these test patterns on several Ntsc THX DVD. For instance, "Toy Story II" in region 1.

DVD Video Essentials

A range of gray test pattern - title 18 chapter 2
A color test pattern - tile 18 chapter 10

DVD AVIA

A range of gray test pattern - title 1 chapter 7
A color test pattern - tile 4 chapter 4
