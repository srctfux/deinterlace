DScaler Help
 

Welcome to the new improved DScaler help files.

The developers have spent a lot of time trying to put as much imformation as we can in this file. The help file is organized in sections:

  1. DScaler Help - This section contains basic information about DScaler including our Licence, installation instructions and keyboard shortcuts.

  2. Features - This section contains additional information about some of DScaler's important features.

  3. Deinterlace Methods - This section contains detailed information about each of DScaler's video deinterlacing methods.

  4. Filters - This section contains detailed information about each of DScaler's video processing filters.

  5. Advanced - This section contains very detailed information about the configuarable parameters in DScaler. This section is intended for advanced users who wish to tweak DScaler.

  6. Developers - This section contains information for those who want to contribute to the DScaler project.
