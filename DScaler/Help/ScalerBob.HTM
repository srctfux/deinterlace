Scaler Bob Method
 

Uses the DirectDraw hardware bob feature (if available), this uses the current field and lets the video card scaler size the picture to fit the screen.  Produces reasonable results for fast moving images.

Who Should Use It?

This method uses very little CPU and is recommended on lower spec machines.

This method should not be used with automatic film detection as the overlay needs to be resized as the type swiches and this will cause flicker.
