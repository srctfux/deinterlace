Electronic Program Guide
 

This DScaler feature allows to display using OSD some information about the TV programmes.
DScaler does not directly collect these data on-line. This task must be done by an external application.

Import an XMLTV file

Using the menu "Import XMLTV File", you can select on your hard drive a XML file and import in DScaler the EPG data contained in that file. That XML file must be compilant with the XMLTV DTD (either 0.5 or 0.6).

To obtain that file, one solution is to use one of the grabbers included in the XMLTV package. A grabber is a script that will access a WEB site proposing the TV programmes, collect data relative to the TV programmes and finally produce a XML file compliant to the XMLTV DTD. This grab can require several hours depending on the number of channels and how many days are grabbed.

One another solution, easier and faster, is to download directly on the WEB that produced file containing EPG data. But of course that file must be available on the WEB for your country (your TV channels).

Import NextViewEPG database

NextViewEPG is an application that collects and displays Electronic Program Guide broadcasted by certain analog TV channels.

Using the menu "Import NextViewEPG database", you can import in DScaler the EPG data collected by NextViewEPG.

Please note that NextViewEPG and DScaler cannot be run at the same time because both of them are controling your TV card. So the idea is to run sometimes NextViewEPG alone to collect data and when it is done, you exit NextViewEPG and run DScaler to import NextViewEPG database. It is not necessary to do that everyday as the EPG providers often distribute programmes for several coming days (often 7 days).

Settings

All those settings are defined in the section "EPG" of the ini file and are available in the Advanced Settings dialog box in DScaler.

Setting Default Description
nxtvepg.exe nxtvepg.exe in DScaler directory This is the path of the NextViewEPG executable.
TimeFrameDuration 1 (hour) This value defines the duration in hours of the time frame used by the programme browser.
DefaultSizePerc 5 This value defines the size of the font used to display EPG in OSD.
MaxCharsPerLine 75 This value defines the maximum number of characters per line for the display of a programme description in the OSD.
ChannelFiltering OFF (0) This boolean defines if the XML file must be filtered to take into account only channels defined in DScaler.
