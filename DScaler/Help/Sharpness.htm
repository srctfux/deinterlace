Sharpness Filter
 

The sharpness filter uses Tom Barry's method for increasing the detail.

This filter should not be used with good sources but it seems to improve detail on VCR sources.

Setting Default Description
Sharpness 128 Amount of sharpness to apply. 0 is none 255 is as much as we can.
Fast Memory Access On This speeds up the filter, but is incompatible with some VIA motherboards.
