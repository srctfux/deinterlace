Program List Dialog
 

How do I find my channels?

For most people setting up the channels should be as simple as selecting a suitable entry from the combo box in Settings, deciding what to set the "Keyed numbers are channels, not positions (US Style)" flag to and pressing scan.

What on earth does "Keyed numbers are channels, not positions (US Style)" mean?

There are basically two ways of thinking about channels. The "US" way where you have a list of frequencies and you refer to each channel by it's position in the list of frequencies or the "European" way where you set up your channels one by one.

With this switch on when you scan the system adds every possible channel to the list and sets the active flag on if the channel has video.

With this switch off when you scan the system only adds channels if the frequency has video.

If you think you can explain this better please let the DScaler team know. We are always on the lookout for better help.

What do all these Controls Do?

The calibration is a procedure using test patterns which allows to find the best video settings for the capture card.

Control What it Does
Name Sets the name of the channel, will be shown on eth menu and OSD when the channel is selected.
Frequency The frequency the tuner is set to.
Set Sets the tuner frequency to whats in the frequency box
The double arrows thing next to Set For fine tuning
Format Sets the format of the channel, for most people this will be the same as the tuner default.
Mute Mutes audio while doing the tuning.
Clear List Deletes all your channels ready for a rescan of if you've messing thing up.

There is no list suitable for where I am?

You could try the new Full Frequency Scan option from the frequencies combo box. This will take a while but should find all the channels.
