Teletext
 

DScaler has a some support for Teletext.

Teletext is a data service provided in the vertical blanking region and is found on TV signals broadcast in 50Hz countries.

Teletext support is still a work in progress and the developers are aware that there are many problems with the current code.

Use the T key to toggle between Normal, Teletext and Mix modes.

To select a page type in the page number while in either Teletext or Mix mode.

DScaler currently supports a few different code pages for different languages. Select the nearest match from the "Datacasting"/"Teletext Code Page" menu.
