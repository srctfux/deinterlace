Channel Setup Catalogue

Introduction
Setting up your local channels can be an annoying process. To try and help we are trying to build up a shared list of channel setups from all over the world.

Please do try and help others by posting your working program.txt files to this mailing list.
How to use
This list is organised around country, then type, then location in alphabetical order. If you find a suitable file please right click on the link then use save as to save the file as program.txt file to your machine. Please ensure that the file is saved as "program.txt". You will then need to copy the file to your DScaler directory which is normally "c:\Program files\DScaler"
