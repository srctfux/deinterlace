Developers 
 

Additional developers are needed for dTV! In a little over a month, this project has garnered a large, loyal, active, and knowledgable user-base of videophiles. This project is making waves in the home theater industry! The project is currently in need of developers with a broad range of skills (C++, assembly, user interface, video internals, etc.). Your skills will be utilized immediately, as we seek to keep this extraordinary momentum going. Please see the Developer Documentation to get started on dTV development and then introduce yourself to the mailing list.

Please refer to the bugs and Todo pages for the current task list. If you think that you can take on a part of one of these tasks, everyone on the mailing list would love to hear from you.

dTV is released under the GNU Public License. Browse the CVS repository and take a look at the code, then download the latest source zipped. The source compiles under MS Visual C++. The application has grown out of the GPL code base of MultiDec and MoreTV, two cable descrambling programs.  By combining the GPL parts of both and extending them both I have been able to create a new legal totally open source project.

The individual source files are in the CVS Repository.  To get check-out and check-in access, you will need to get cvs and to make a few configuration changes see Win32-CVS-SSH Guide and Win32-CVS-SSH HowTo for more details.
