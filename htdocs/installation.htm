Installation & Setup 
 

Hardware Prerequisites

See the supported cards page to find out if your video capture card is supported. Please note that currently, your performance may degrade significantly if your CPU is slower than 400 MHz or your video card is not AGP. In addition, when using a system that is borderline fast enough, dTV is sensitive to system tray applications and active services.


Installation

Unzip all the files into a directory and run dTV.exe.


Getting It Working

PAL

  1. Delete any previous copy of dTV.ini.
  2. Select video card  and tuner type from dialog box that appears on startup.  If your exact card type is not listed then try Standard Card (bt878).  The make of tuner should be on the silver box on the card, choose the correct type depending on your region.
  3. Select required input from Settings/Video Input.
  4. Adjust Video settings using Setting/Video Adjustments

NTSC

  1. Delete any previous copy of dTV.ini.
  2. Select video card and tuner type from dialog box that appears on startup. If your exact card type is not listed then try Standard Card (bt878).  The make of tuner should be on the silver box on the card, choose the correct type depending on your region.
  3. Select NTSC from Settings/Video Format.
  4. Select required input from Settings/Video Input.
  5. Adjust Video settings using Setting/Video Adjustments

For more information about the advanced features available via the dTV.ini file, please see the .ini documentation.


What if That Doesn't Work?

Please visit the HTPC section of the AVS Forum for help. The forum has high traffic and a good search function. Chances are high that a search will reveal that others have had your problem too and may have figured out a way to fix it.
