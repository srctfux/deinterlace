Requirements:

I think it will run on Win 95/98 and NT. I have tested it on 98, Windows 2000.

You will need a processor faster than about Pentium II 400MHz.

You will definitely need a PCI TV card with a BT848 BT849 BT878 BT879 in it.

This software will not work with ATI All-in-Wonder or Matrox TV cards.

Warning

This program installs a temporary driver while it runs.  This means that some of the code runs inside Ring0 along with the kernel.  What this means is that if anything goes wrong this program can crash your whole system causing a BSOD in NT or a hanging 95/98.  I have had no problems running the exe stand alone but you may get problems.  If using a debugger then it it is very important that the program is allowed to shut down normally.  If you use Stop Debugging while running the program you system WILL crash.

As a precaution sync the disks before running this program.  Use the sync utility from http://www.sysinternals.com/ for whichever operating system you are running.

Installation:

Put all the files into a directory and run dTV.exe.

Getting It Working:

On my setup I have got it working with PAL and NTSC.  My card has a BT878 and so it may work differently or not at all with your card.

I have not really tested the tuner part of the program, so first select the external input

PAL Setup

  1. Select video card type from list in Hardware/TV-Card
  2. Select tuner type from list in Hardware/Select Tuner
  3. In Hardware/PLL settings Box marked PLL (28MHz Crystal) should be ticked if you have anything except an B848 card, try with first
  4. Select required input from Settings/Video Input.
  5. Adjust Video settings using Setting/Video Adjustments

NTSC Setup

  1. Select video card type from list in Hardware/TV-Card
  2. Select tuner type from list in Hardware/Select Tuner
  3. Select NTSC from Settings/Video Format.
  4. Select required input from Settings/Video Input.
  5. Adjust Video settings using Setting/Video Adjustments

Try the various PLL settings.
E-Mail me for help so that we can try and get your card supported

What does it do?

At the moment it allows you to trawl thought the available pull down modes manually using the space bar.  So to use get a picture by playing around with the settings as above.  Then put in a film and press the spacebar until the weave patterns disappear.

It is possible to specify a bTV plugin for the program to use.  The default is bDeinterlace.dll with UseKey1 set to 1.

History

1.1 Added support for bTV plugins and full screen support.
1.0 The original version only had support for manual inverse telecine.
