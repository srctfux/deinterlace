How To Use

  1. Downlaod the plug-in and place in bTV's plugins directory.
  2. Start bTV press F4 to go to Hardware Settings.
  3. Make sure Enable flipping is on
  4. Stop and restart bTV
  5. Press F7 to go to Decoding screen and select PALMovie plug-in and make active.
  6. Experiment with the Key1 button to see which one eliminated interlace.
  7. Sit back and enjoy the film....
