About 
 

dTV is an open-source, SourceForge hosted project to create a program to deinterlace TV or Video through a computer without the need for an expensive processor box.

The spiritual home of this project is the Home Theater Computers Section on the AV Science Forum.

This project is in the early stages.  If you are interested at all in this project then join the Mailing List and say hello.

dTV has grown out of the GPL code base of MultiDec and MoreTV, two cable descrambling programs.  By combining the GPL parts of both and extending them both, John Adcock was able to create a new legal totally open source project.

Please see the downloads page to download the latest version.
